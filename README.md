<h1>OMS test assignment</h1>

<h5>To run this app you need:<h5>

1. `yarn build`
2. `yarn typeorm:run`
3. `yarn seed:run -s CreateProducts` (or just `yarn seed:run` if want to seed all entities at once)
4. `yarn start:dev`

> Note:  all seeded employees have password 123456
